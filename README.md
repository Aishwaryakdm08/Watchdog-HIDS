# Watchdog - Host-Based Intrusion Detection System (HIDS)

## Overview
Watchdog is a Python-based Host-Based Intrusion Detection System that analyzes system logs to detect suspicious activities and security threats.

## Objectives: 
- Automatic system log monitoring and analysis 
- Detection of suspicious system and security events 
- Identification of potential intrusion attempts 
- Alert generation with severity levels  
- Interactive dashboard with charts and recent alerts 
- Email notifications for critical security events 

## Tools and technologies: 
- Frontend: HTML, CSS, JavaScript 
- Backend: Python with Flask Framework 
- Database: SQLite 
- Data Visualization: Chart.js 
- Development Tool: Visual Studio Code 

## Methodology: Incremental Model 
The project is developed using the Incremental Model. The system is built 
in small increments, where each increment adds new functionality. 
Initially, log collection is implemented, followed by log analysis and 
 
 
intrusion detection. Database storage and dashboard development are 
added in later increments. Additional features such as data visualization 
and email alert notifications are integrated in later stages. Each increment 
is tested before moving to the next stage. 
 
### Phases and Increments - 
- Log collection from the Windows event logs 
- Log analysis using rule-based pattern matching 
- Intrusion detection logic implementation 
- Database storage of alerts and log data 
- Dashboard development with charts and reports 
- Testing, monitoring, and system refinement 
 
## Modules:
